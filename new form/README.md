# Animated Sign Up / Sign In Form

A modern and animated Sign Up / Sign In user interface built with HTML, CSS, JavaScript, and GSAP. The project features smooth transitions between authentication forms and a clean, responsive-inspired design.

## ✨ Features

- Modern Sign Up interface
- Animated Sign In interface
- Smooth transitions using GSAP
- Clean and user-friendly design
- Background image panel
- Pure HTML, CSS, and JavaScript implementation

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- GSAP (GreenSock Animation Platform)

## 🎯 How It Works

- The application initially displays the Sign Up form.
- Clicking the Sign In button triggers:
  - A sliding animation of the right panel.
  - The Sign Up form fades out.
  - The Sign In form fades in.
  - The title, description, and button text are updated dynamically.
- Clicking the button again switches back to the Sign Up form.
